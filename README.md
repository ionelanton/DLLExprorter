DLLExprorter
============

Exprort methods to unmanaged code in C#

## Command-line arguments:
```
DLLExporter.exe -in:X:\library.dll [-out:X:\some_folder\library.dll] [-noclear]
```
* -in:		Input dll file
* -out:		Output dll file
* -noclean	Don't remove *.il adn *.res from output folder
